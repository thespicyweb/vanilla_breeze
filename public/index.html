August 2025 Update: Vanilla Breeze is shutting down by the end of the year!
Tailwind 4 has largely eliminated the need to use a tool like this. It's a major rewrite oriented around real CSS variables and a CSS-based config. This a welcome development, and while technically there's still a use case for wanting to convert HTML-based utility classes to a stylesheet, we expect most Tailwind 4+ users will either author CSS using the new CSS variables from scratch, or use off-the-shelf TW-based components which are still using utility classes and not care. Either way, Vanilla Breeze was a fun hack and served its purpose in the pre-v4 days. Cheers! -Jared
